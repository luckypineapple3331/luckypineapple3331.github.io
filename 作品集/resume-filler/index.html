简历自动填充助手

简历信息一次录入，在任意招聘官网一键自动填充表单，告别重复复制粘贴，提升校招投递效率。

Chrome 扩展 Manifest V3 模糊关键词匹配 本地数据存储
⚡ 核心功能
  • ⚡
    一键填充检测到招聘页面后，点击按钮自动将预存简历信息填入页面表单，支持腾讯、字节、网易等各大校招官网。
  • 🔍
    模糊识别通过关键词相似度算法识别不同网站的表单字段，无需针对每个网站单独配置，通用性强。
  • 🔒
    本地存储 + 密码保护所有简历数据仅存储在本地浏览器，不上传任何服务器；可设置解锁密码，防止他人查看。
  • 💾
    数据导出支持将简历数据导出为 JSON 文件备份，换电脑时可直接导入恢复。
📋 支持填充的信息板块
基础信息
姓名、出生年月、联系电话、邮箱、地址
教育经历
学校、专业、学历、就读时间、主修课程（支持多条）
实习经历
公司、岗位、实习时间、工作内容（支持多条）
项目经历
项目名称、角色、时间、描述（支持多条）
校园经历
经历名称、职务、任职时间、描述（支持多条）
获奖经历
奖项名称、时间、角色、描述（支持多条）
技能证书
工具技能、语言能力、其他证书
个人简介
自我介绍 / 个人评价
📦 安装方法
1
打开 Chrome 浏览器，在地址栏输入 chrome://extensions
2
开启右上角的「开发者模式」开关
3
点击「加载已解压的扩展程序」，选择 resume-filler 文件夹
4
安装成功后，浏览器右上角工具栏会出现 📄 图标，点击即可使用
