# 简历自动填充助手

> 一款 Chrome 浏览器扩展，将简历信息预先填好，在任意招聘官网一键自动填充表单，告别重复复制粘贴。

---

## 功能介绍

### 核心功能

- **一键填充**：检测到招聘页面后，点击「⚡ 一键填充」按钮，自动将预存简历信息填入页面表单
- **模糊识别**：通过关键词相似度算法识别不同网站的表单字段，无需针对每个网站单独配置
- **本地存储**：所有简历数据仅存储在本地浏览器，不上传任何服务器

### 支持填充的信息板块

| 板块 | 包含字段 |
|------|----------|
| 基础信息 | 姓名、出生年月、联系电话、电子邮箱、联系地址 |
| 教育经历 | 学校名称、专业、学历、就读时间、主修课程（支持多条） |
| 实习经历 | 公司名称、岗位名称、实习时间、工作内容（支持多条） |
| 项目经历 | 项目名称、担任角色、项目时间、项目描述（支持多条） |
| 校园经历 | 经历名称、担任职务、任职时间、职务描述（支持多条） |
| 获奖经历 | 奖项名称、获奖时间、担任角色、奖项描述（支持多条） |
| 技能证书 | 工具技能、语言能力、其他证书 |
| 个人简介 | 自我介绍/个人评价 |
| 作品集 | 个人主页、作品链接、GitHub 地址等 |

### 其他功能

- **密码保护**：可设置解锁密码，防止他人在你的电脑上查看简历信息
- **数据导出**：将简历数据导出为 JSON 文件备份
- **校园经历兜底**：若招聘网站没有"校园经历"专属板块，校园经历条目会自动合并到项目经历中填写

## 安装方法

本插件为本地开发版，需手动加载到 Chrome 浏览器。

1. 打开 Chrome 浏览器，在地址栏输入：
   ```
   chrome://extensions
   ```

2. 开启右上角的「**开发者模式**」开关

3. 点击「**加载已解压的扩展程序**」

4. 选择本项目的文件夹（即包含 `manifest.json` 的那个文件夹）

5. 安装成功后，浏览器右上角工具栏会出现 📄 图标

> **提示**：如果工具栏没有显示图标，点击拼图形状的「扩展程序」按钮，找到「简历自动填充助手」并固定到工具栏。

## 使用方法

### 第一步：录入简历信息

1. 点击浏览器右上角的 📄 图标，打开插件弹窗
2. 在「📋 简历信息」标签页中，依次填写各板块内容
3. 对于教育、实习、项目等多条目板块，点击「+ 新增××经历」按钮可添加多条记录
4. 填写完成后，点击底部「💾 保存信息」按钮

> **时间字段说明**：各板块中的开始/结束时间字段仅作为你本人的参考记录，**填充时不会自动写入页面**（因各网站时间格式差异较大），请在一键填充完成后手动填写时间。

### 第二步：在招聘页面填充

1. 打开任意招聘官网，进入简历填写页面（如腾讯、字节、网易等校招页面）
2. 点击浏览器右上角 📄 图标
3. 若顶部出现「✅ 检测到招聘页面」提示，点击「**⚡ 一键填充**」按钮
4. 等待填充完成，查看「共填充 N 个字段」的结果提示
5. 检查页面，**手动补充时间字段**及未能自动识别的字段

> **若未检测到招聘页面**：顶部会显示「⚠️ 未检测到招聘页面」，可点击「🔍 手动识别」强制触发填充。

### 修改已保存的信息

- 直接在弹窗中修改对应字段，再点击「💾 保存信息」即可覆盖
- 点击「↩ 重置」可将当前弹窗内容还原为上次保存的状态（不会删除已保存数据）

## 设置说明

点击顶部「⚙️」按钮或切换到「⚙️ 设置」标签页：

### 密码保护

1. 打开「启用密码保护」开关
2. 输入并确认密码（4-16 位）
3. 点击「保存密码」
4. 之后每次打开插件弹窗时，需先输入密码解锁

### 数据管理

- **导出简历数据**：将当前所有简历信息导出为 `.json` 文件，可用于备份或迁移
- **清除所有数据**：删除本地存储的全部简历信息（操作不可撤销，请谨慎使用）

---

## 注意事项

- 填充结果取决于目标网站的页面结构，部分字段可能无法自动匹配，需手动补充
- 时间字段（开始/结束时间）需在填充完成后手动填写
- 如遇网站页面更新导致填充失效，可尝试点击「🔄 重新填充」按钮
- 所有数据存储于浏览器本地（`chrome.storage.local`），清除浏览器数据时会一并删除，建议定期导出备份

```
├── popup.js            # 弹窗交互逻辑
├── popup.css           # 弹窗样式

    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```
